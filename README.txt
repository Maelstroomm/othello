Ce projet à été réalisé en binôme dans le cadre du cours d'informatique de 2ème année de la Prépa de INP.

Le Jeu d'Othello est un jeu de stratégie très populaire dans lequel deux joueurs s'affrontent en plaçant leurs pions sur un plateau afin de retourner les pions adverses et d'en capturer un maximum. 
Dans le cadre de ce projet, nous avons développé une application en python permettant de jouer à ce jeu contre l'ordinateur avec une stratégie de base et nous avons ensuite amélioré cette application avec des options supplémentaires.
Pour améliorer la stratégie de jeu de base, nous avons exploré différentes méthodes telles que l'algorithme minmax, la coupure alpha-bêta et les méthodes probabilistes. 
Nous avons effectué des tests pour évaluer la performance de ces différentes stratégies.
